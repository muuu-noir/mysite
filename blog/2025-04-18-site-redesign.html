サイトデザインリニューアルしました

リニューアルの背景

以前のサイトからデザインを一新し、より洗練されたポートフォリオサイトへと生まれ変わりました。

今回のリニューアルでは、グラスモーフィズムとミニマルデザインを基調に、「確かなロジック、時間を愛でる感性」というコンセプトを視覚的に表現することを目指しました。

デザインのポイント

1. グラスモーフィズム

すりガラスのような透明感のあるカードデザインを採用。背景のぼかしと半透明の境界線が、奥行きと軽やかさを演出します。

2. ダークモード対応

システム設定に連動した自動切り替えに加え、手動でのテーマ切り替えにも対応。どちらのモードでも美しく表示されるよう、カラーパレットを細かく調整しました。

3. アニメーション

スクロールに連動したリビールアニメーション、ヒーローセクションのパララックス効果、文字のスプリットアニメーションなど、さりげないモーションで心地よい体験を提供します。

4. Web Components

<site-header>、<site-footer>、<glass-card> など、再利用可能なカスタム要素でコードの保守性を向上させました。

技術スタック

  • HTML5 + CSS3（カスタムプロパティ、グリッド、Flexbox）
  • Vanilla JavaScript（Web Components、IntersectionObserver）
  • Google Fonts（Outfit、Inter）
  • GitHub Pages でホスティング

今後の展望

引き続き、コンテンツの充実とパフォーマンスの最適化を進めていきます。ブログ機能の追加も計画中です。
