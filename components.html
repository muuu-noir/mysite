Glassmorphism

半透明背景 + ブラー + ボーダーによるガラス効果。ダークモードで自動的に配色が切り替わる。

Glass Card

背景が透過して見えるガラスモーフィズム効果。hover時にわずかに浮き上がる。

/* Glass Variables */
--glass-bg: rgba(255,255,255,0.65);
--glass-border: rgba(255,255,255,0.4);
--glass-blur: blur(25px);
--glass-shadow: 0 10px 40px rgba(31,38,135,0.03);

.glass-card {
  background: var(--color-surface);
  border: 1px solid var(--color-border);
  box-shadow: var(--glass-shadow);
  border-radius: 20px;
  padding: var(--sp-card-p);
}

Reveal Animation

IntersectionObserverによるスクロールトリガーアニメーション。要素がビューポートに入ると、下から上にフェードイン。

この要素は .reveal.active — 表示済み状態。

.reveal {
  opacity: 0;
  transform: translateY(30px);
  transition: 1.0s cubic-bezier(0.2, 1, 0.2, 1);
}
.reveal.active {
  opacity: 1;
  transform: translateY(0);
}

/* Usage: add reveal attribute to components */
<glass-card reveal>...</glass-card>
<section-title title="..." reveal></section-title>

Dark Mode

ヘッダーのトグルボタンでライト/ダークモードを切り替え。CSS変数が一括で切り替わり、全コンポーネントが自動対応。prefers-color-schemeにもフォールバック対応。

トークンLight → Dark
--color-bg#F9FAFC → #0B0E14
--color-surface#FFFFFF → #1A1F2B
--color-text-main#1D1D1F → #E2E8F0
--color-text-sub#6E6E73 → #94A3B8
--glass-bgrgba(255,255,255,0.65) → rgba(26,31,43,0.7)
/* Toggle via data-theme attribute */
:root[data-theme="dark"] {
  --color-bg: #0B0E14;
  --color-surface: #1A1F2B;
  --color-text-main: #E2E8F0;
  ...
}

/* System preference fallback */
@media (prefers-color-scheme: dark) {
  :root:not([data-theme="light"]) { ... }
}
